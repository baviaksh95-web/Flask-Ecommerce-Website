# 🛒 Flask E-Commerce Website

A simple E-Commerce web application built using Flask, SQLAlchemy, Bootstrap, and SQLite.

## 🚀 Features

- User Registration & Login (Flask-Login)
- Category-based product filtering
- Product detail page
- Add to Cart functionality
- Remove from Cart
- Search products
- Admin Add Product feature
- Responsive UI using Bootstrap

## 🛠️ Tech Stack

- Python
- Flask
- Flask-Login
- SQLAlchemy
- SQLite
- HTML5
- CSS3
- Bootstrap

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/Flask-Ecommerce-Website.git
```

2. Navigate into project

```
cd Flask-Ecommerce-Website
```

3. Create virtual environment

```
python -m venv venv
```

4. Activate virtual environment (Windows)

```
venv\Scripts\activate
```

5. Install dependencies

```
pip install -r requirements.txt
```

6. Run the app

```
python app.py
```

App will run on:
```
http://127.0.0.1:5000/
```
